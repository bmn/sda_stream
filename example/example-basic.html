sda_stream Basic Example

Here's the quickest way of getting sda_stream going:

  • Rename 
    config.js.dist
    to 
    config.js
    .
  • Add all the channels you want to follow on Ustream to that file.
  • Get your Ustream API key from the Ustream Developer Network, if you haven't already, and put that in as well.
  • Get a webpage and make sure jQuery 1.4, 
    config.js
    and 
    listing.js
    are being called.
  • Under all your HTML, run 
    SDAStream()
    in Javascript (protip: assign it to a variable and keep Britain tidy!)

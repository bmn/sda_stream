sda_stream PHP Request Example

By default Ustream only allows 5000 API requests per day, and a maximum of 10 channel checks per request. If you have a lot of users you'll reach 5000 quickly.

Also, having your API key publicly available makes it vulnerable to use by others. So it's best to use a server-side language to make your requests.

By telling SDAStream to use the included PHP script, you hide your key and can also have it store the result for a set period (default 1 minute). This saves on requests and speeds up your app.

  • Make sure 
    config.js
    , if it exists, doesn't contain any channels or your API key.
  • Rename 
    config.php.dist
    to 
    config.php
    , and add your channels and key to it.
  • Use the Javascript SDAStream as normal, but add the option 
    php
    which should point to the sda_stream directory.

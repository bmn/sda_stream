sda_stream Multiple Instances Example

It's possible to run more than one SDAStream() instance at the same time:

  • You can set up one instance maximum in the same way as in the basic example, but this isn't required.
  • When calling 
    SDAStream()
    , pass it an object containing your settings for any of 
    channels
    , 
    key
    , 
    skin
    and 
    selectors
    .
